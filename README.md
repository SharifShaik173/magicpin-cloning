# magicpin Homepage Clone
 
A front-end recreation of the **magicpin** ("India's #1 Savings App") homepage, built with plain HTML and CSS.
 
## 🔗 Preview
 
Open `index.html` in your browser to view the page locally.
 
## 📁 Project Structure
 
```
.
├── index.html          # Main markup for the homepage
├── style.css           # All styling for the page
└── Assets/             # Local images/icons used by the page (logo, illustrations, banners, etc.)
```
 
> **Note:** The `Assets/` folder is referenced throughout `style.css` and `index.html` (e.g. `logo.png`, `homepage-illustration.svg`, `footer-bg.svg`, category icons, brand banners). Make sure this folder is included alongside the HTML/CSS when cloning or deploying, or update the paths if you rename/move it.
 
## ✨ Features / Sections
 
- **Navigation bar** — logo, nav links (About us, Around you, Blog, Velocity, Vera, Business), app download links, and a Sign In/Up button
- **Hero section** — gradient/illustration background with headline and location + search inputs
- **Category strip** — horizontally scrollable category icons with a "View All Categories" link
- **Top brands showcase** — scrollable rows of top Food Delivery, Fashion, and Online brand logos linking to merchant pages
- **Promo banners** — image-based promotional sections
- **Business CTA cards** — "magicpin for Brands" and "magicpin for SME retailers" call-to-action cards
- **Footer** — social links, category shortcuts, company/legal info, popular cities/localities/restaurants/fashion hubs, site navigation links, and app store badges
## 🛠️ Built With
 
- **HTML5**
- **CSS3** (Flexbox, Grid, custom scroll containers, hover transitions, border-radius shaping)
- [Font Awesome](https://fontawesome.com/) (via CDN) for icons
## 🎨 Styling Notes
 
- Primary brand colors: `#431cc9` (purple), `#ef1c71` / `#e8197d` (pink/magenta)
- Font family: `Gill Sans`, `Gill Sans MT`, `Calibri`, `Trebuchet MS`, sans-serif
- Horizontally scrollable sections use `overflow-x: scroll` with hidden scrollbars for a clean carousel look
## 🚀 Getting Started
 
1. Clone this repository
```bash
   git clone https://github.com/<your-username>/<repo-name>.git
```
2. Navigate into the project folder
```bash
   cd <repo-name>
```
3. Open `index.html` in your browser (or use a live server, e.g. the VS Code "Live Server" extension)
